# 俄罗斯方块（Tetris）

一个**无需依赖、离线可玩**的俄罗斯方块：原生 JavaScript + Canvas。

## 运行

- 直接用浏览器打开 `index.html` 即可开始游戏。

如果你更喜欢用本地静态服务器（可选）：

```powershell
cd tetris
python -m http.server 5173
```

然后打开 `http://localhost:5173`。

## 协作开发

本项目使用 Git 管理。克隆后可以自由修改并提交 PR。

## 操作

- **←/→**：左右移动
- **↓**：加速下落（按住）
- **↑**：旋转
- **Space**：瞬间落地
- **P**：暂停/继续
- **R**：重开

## 规则与特性

- **7-bag 随机**：每 7 个方块包含 I/O/T/S/Z/J/L 各一个，分布更均匀
- **幽灵落点**：显示当前方块落地位置
- **计分**：
  - 软降每格 +1
  - 硬降每格 +2
  - 消行按等级加成（1/2/3/4 行对应 100/300/500/800 * level）
- **等级**：每 10 行升 1 级，自动加快下落速度

## 文件结构

| 文件 | 用途 | 改外观/逻辑时 |
|-----|------|-------------|
| `index.html` | 页面结构 | 增删按钮、文字等 |
| `style.css` | 外观样式 | **改颜色、布局、字体等** |
| `main.js` | 游戏逻辑 | 改玩法、计分、按键等 |
